---

## ⚡ Installation & Setup

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Environment Variables (.env):**

   ```env
   DATABASE_URL=your_railway_mysql_url
   DB_SSL=true
   BLOB_READ_WRITE_TOKEN=your_vercel_blob_rw_token
   ```

3. **Run locally:**

   ```bash
   npm run dev
   ```

4. **Visit:** `http://localhost:3000`
